#  ***Проект: Mesto-react ***

## Сайт - интерактивная страница, куда можно добавлять фотографии, удалять их, ставить лайки, редактировать профиль.

___
### **Тут используются технологии:**
+ ***Flex-box***
+ ***Grid Layout***
+ ***БЭМ Nested***
+ ***Адаптивная верстка***
+ ***React***
+ ***UseState***
+ ***UseEffect***
___
